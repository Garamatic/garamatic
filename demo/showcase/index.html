⚡ Event Flow Visualization

Portal
→
Gatekeeper
→
RabbitMQ
→
Ticket Masala
→
Mailing
→
Mailhog
A citizen submits a ticket via the Portal → Gatekeeper validates & publishes a ticket.created event to RabbitMQ → Ticket Masala processes and stores it → Mailing Service sends confirmation → captured in Mailhog for inspection.
Agentic
→
LLM (Ollama)
→
Ticket Masala
→
Odoo Bridge
→
Odoo ERP
The Agentic Service queries the local LLM for intelligent ticket routing → updates Ticket Masala → Odoo Bridge syncs invoice data to Odoo ERP.
